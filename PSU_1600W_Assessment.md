# ESC4000G4 电源评估: 1600W 1+1 冗余

> 电源: 1+1 Redundant 1600W 80 PLUS Platinum  
> 负载目标: 3x NEC VE 1.0 + 1x Intel Xeon Phi 7120P  
> 评估时间: 2026-05-19

---

## 1. 电源规格确认

| 参数 | 数值 |
|------|------|
| 配置 | 1+1 Redundant (双模块热插拔冗余) |

| 总冗余容量 | 1600W (不是 3200W!) |
| 效率等级 | 80 PLUS Platinum |

| 输入电流 | 9.5A per module |

### 1.1 1+1 冗余的工作原理

```
正常模式 (双 PSU 健康):
┌──────────┐      ┌──────────┐
│ PSU A    │      │ PSU B    │
│ 1600W    │      │ 1600W    │
│ 输出 800W │◄────►│ 输出 800W │
└────┬─────┘      └────┬─────┘
     │                 │
     └───────┬─────────┘
             ▼
         系统负载
         1600W

故障模式 (PSU A 故障):
┌──────────┐      ┌──────────┐
│ PSU A    │      │ PSU B    │
│ ❌ 离线   │      │ 1600W    │
│          │      │ 输出 1600W│
└──────────┘      └────┬─────┘
                       │
                       ▼
                   系统负载
                   1600W
```

**关键理解**: 1+1 冗余的可用功率是 **1600W**，不是 3200W。两个 PSU 是备份关系，不是叠加关系。

## 3. 电源能力分析

### 3.1 正常双 PSU 运行

| 参数 | 数值 |
|------|------|
| 系统 DC 负载 | 1730W |
| 单 PSU 输出 | 1730W / 2 = **865W** |
| 单 PSU 负载率 | 865W / 1600W = **54%** |
| Platinum @ 54% 效率 | ~93% |
| 单 PSU AC 输入 | 865W / 0.93 = **930W** |
| 总 AC 输入 | 930W × 2 = **1860W** |
| 输入电流 @ 220V | 1860W / 220V = **8.5A** |
| 输入电流 @ 200V | 1860W / 200V = **9.3A** |

**结论**: 正常双 PSU 运行时，单 PSU 54% 负载，在 Platinum 效率甜点区（40-60%），**完全没问题** ✅

### 3.2 单 PSU 故障 (冗余切换)

| 参数 | 数值 |
|------|------|
| 剩余 PSU 输出 | **1730W** |
| 单 PSU 负载率 | 1730W / 1600W = **108%** |
| Platinum @ 108% 效率 | ~89% (过载区) |
| 单 PSU AC 输入 | 1730W / 0.89 = **1944W** |
| 输入电流 @ 220V | 1944W / 220V = **8.8A** |
| PSU 额定电流 | 9.5A |
| 电流余量 | 9.5A - 8.8A = **0.7A (7.4%)** |

**结论**: 
- 输出侧: 1730W > 1600W 额定 = **过载 8%**
- 输入侧: 8.8A < 9.5A 额定 = 勉强够
- **冗余切换后 PSU 处于过载状态** ⚠️

### 3.3 PSU 过载的后果

| 过载程度 | 预期行为 |
|---------|---------|
| 100-105% | PSU 风扇全速 (>15,000 RPM)，噪音 >85dB |
| 105-110% | 温度急剧上升，可能触发 OTP (过温保护) |
| 110-120% | PSU 可能自动关机保护，系统断电 |
| >120% | OPP (过功率保护) 立即切断输出 |

**实际风险**: PSU 在 108% 负载下：
- 大概率可以运行（服务器 PSU 通常有 10-15% 超额能力）
- 但温度极高，风扇全速
- 长期运行会严重缩短 PSU 寿命（电容老化加速）
- **随时可能触发 OTP/OPP 保护而宕机**

## 4. 不同负载场景下的电源状态

| 场景 | DC 负载 | 双 PSU 单载 | 单 PSU 单载 | 冗余可行性 |
|------|---------|------------|------------|-----------|
| 待机 | 450W | 28% ✅ | 56% ✅ | 完全冗余 |
| 轻载 (1 VE + Phi) | 750W | 47% ✅ | 94% ⚠️ | 勉强冗余 |
| 中载 (2 VE + Phi) | 1300W | 81% ✅ | **163% ❌** | **冗余失效** |
| 满载 (3 VE + Phi) | 1730W | 54% ✅ | **108% ❌** | **冗余失效** |

### 关键结论

```
3 VE + 7120P 满载 = 1730W
1600W PSU × 1 = 1600W
1730W > 1600W → 冗余在单 PSU 故障时无法维持
```

**即使 PSU 有 10% 超额能力 (1760W)，也处于极限边缘。**

---

## 5. 额外风险因素

### 5.1 PSU 老化

- 1600W PSU 使用 3-5 年后，实际容量可能衰减到 1400-1500W
- 老化后的 PSU 在 1730W 负载下几乎一定会触发保护

### 5.2 环境温度

- PSU 额定功率在 25°C 环境下标定
- 机房温度 > 35°C 时，PSU 需要降额使用（可能只有 1400W）
- 4 张加速卡在 2U 机箱内会加热整个系统

### 5.3 电网波动

- 输入电压低于 200V 时（如 190V），PSU 需要更大电流
- 9.5A × 200V = 1900W 输入极限
- 实际负载 1944W > 1900W → 可能触发保护

### 5.4 启动浪涌

- 系统开机时，所有风扇和硬盘同时启动
- 启动浪涌可能比稳态高 10-20%
- 如果 PSU 没有足够的浪涌余量，可能无法开机

## 6. 解决方案

### 方案 A: 降级配置 (推荐)

将系统功耗降到 1600W 以内：

| 配置 | DC 负载 | 冗余可行性 |
|------|---------|-----------|
| **2 VE + 7120P** | ~1200W | ✅ 单 PSU 75%，冗余有效 |
| **3 VE (不加 Phi)** | ~1300W | ✅ 单 PSU 81%，冗余有效 |
| 2 VE + 5110P (225W) | ~1125W | ✅ 单 PSU 70%，冗余有效 |

### 方案 B: 升级电源

| 升级选项 | 功率 | 满载单载 | 冗余可行性 | 成本 |
|---------|------|---------|-----------|------|
| 2000W 1+1 冗余 | 2000W | 87% | ✅ 有效 | 中 |
| 2200W 1+1 冗余 | 2200W | 79% | ✅ 有效 | 中高 |
| 2000W+1600W 混合 | 2000W | 87% | ✅ 有效 | 低 |

**推荐**: 升级到 **2000W 1+1 冗余 PSU**

### 方案 C: 限制功耗

通过软件限制加速卡功耗：

```bash
# VE 卡功耗限制 (如支持)
# Phi 功耗限制
micctrl --bootp 200W mic0  # 将 7120P 限制在 200W
```

**缺点**: 性能下降，7120P 本就不强，再降功耗更弱。

## 7. 最终结论

### 1600W 电源 + 3 VE + 7120P = ⚠️ 不推荐

| 维度 | 评估 |
|------|------|
| 正常运行 | ✅ 双 PSU 各 54%，完全没问题 |
| 冗余切换 | ❌ 单 PSU 108% 过载，冗余失效 |
| PSU 寿命 | ❌ 长期满载加速老化 |
| 稳定性 | ❌ 随时可能触发 OTP/OPP 保护 |
| 噪音 | ⚠️ 满载时风扇 >15,000 RPM，>85dB |

### 明确结论

```
1600W PSU 的额定容量: 1600W
3 VE + 7120P 的峰值功耗: ~1730W

1730W > 1600W → 超出 8%

这意味着:
1. 两个 PSU 都健康时 → 可以运行
2. 一个 PSU 故障时 → 系统大概率宕机
3. 冗余电源失去了冗余意义
```

### 建议

| 你的情况 | 建议 |
|---------|------|
| **坚持 3 VE + 7120P** | **必须升级 PSU 到 2000W+** |
| 不想升级电源 | **改为 2 VE + 7120P** (负载 ~1200W，冗余有效) |
| 追求最大算力 | **3 VE 不加 Phi**，等第四张 VE |
| 已有 7120P 想物尽其用 | 单独找个机箱装，不要和 VE 抢电源 |
